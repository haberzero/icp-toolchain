1. 程序初始化流程

3. 数据/实例流转过程

来构建工程的依赖关系模型。

**重要提示**：
- 基础层文件（被其他文件依赖）的 dependent_relation 应该为空列表 []
- 业务层和应用层文件应该依赖基础层文件
- 依赖方向从上层指向下层，确保单向依赖，避免循环依赖
- 所有依赖路径必须在工程文件夹结构中真实存在，你可以参考上面的完整路径列表
